# Ghidra Scripts
Open sourced Ghidra scripts by *NikOffSec*


All scripts written in PyGhidra for modern Python libraries and features from CPython. Make sure to launch ghidra as pyghidra to use.
Documentation for scripts can be found in their respective directories to make customization
as seamless as possible. Please refer to https://ghidra.re/ as an extensive reference for ghidra
script development. Good luck in your reversing jouney!
